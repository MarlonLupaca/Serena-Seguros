# Documentacion del proyecto

Esta carpeta contiene la documentacion tecnica del proyecto en formato Word.

## Archivo

- `Documentacion_Serena_Seguros.docx` — documento principal listo para leer.

## Que contiene

1. Vision general del proyecto y los 5 portales
2. Stack tecnologico (backend, frontend, BD)
3. Estructura del proyecto
4. Como levantar el proyecto en local (incluida la migracion de la Fase 7)
5. Esquema de base de datos (las 23 tablas por dominio)
6. Convenciones de codigo
7. Autenticacion y permisos
8. Estado por portal: que vistas funcionan, cuales son placeholder, deudas
9. Endpoints disponibles agrupados por modulo
10. Datos demo del seeder
11. Roadmap por fases (0 a 7)
12. Deudas tecnicas y pendientes
13. Troubleshooting comun

## Mantenimiento

El contenido de este Word se mantiene en sintonia con `DESARROLLO.md` en
la raiz del proyecto. La documentacion se actualiza al cierre de cada fase.
